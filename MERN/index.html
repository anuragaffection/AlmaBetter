What is Internet
The internet is a network of network. It connect all the devices globally using standard protocol .

What is website ?
Website is a collection or web pages or a web page where the some information is available and have unique domain name.

What is Web App ?
Web app is also a website with dynamic in nature where users can interact .

What is Full stack Developer
A person who works on both Frontend and Backend side to develop or build a website or web apps .

What is Frontend Developer ?
A person who works only on Frontend side to develop a website or web apps. They generally use technologies like HTML, CSS & JAVASCRIPT

What is Backend Developer ?
A person who works majorly on Backend side to write or develop website . They generally use technologies like Node js, Java, Python, & alos databases.

What is MERN ?
MERN is a technologies stack which is used to build a web apps. It consists of M- MongoDB, E - ExpressJS, R - ReactJS, N - NodeJs
