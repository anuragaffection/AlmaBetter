Data Language {
    -- it refers to commands that interact with database
    -- DDL defines schema, 
    -- DML manipulates data, 
    -- DCL controls permissions, 
    -- TCL manages transactions.

    -- ALTER

    03. A Database Management System (DBMS) is a ___________ (software / framework) that enables users to interact with a database.
    
    04. In a ___________ (hierarchical / network) DBMS, data is organized in a tree-like structure
    
    05. ___________ (distributed / network) DBMS is based on the mathematical model of directed graphs.


    Answers
    01. server
    02. logic
    03. software
    04. hierarchical
    05. network
