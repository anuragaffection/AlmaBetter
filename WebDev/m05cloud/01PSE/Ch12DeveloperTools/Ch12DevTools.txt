    01. ___________(IDE/VCS) is a software application that provides tools and features for coding, debugging, and testing software in a unified environment.
    
    02. IDEs often integrate with version control systems like  _________(Atom/Git), allowing developers to manage code changes, collaborate with team members, and track the history of code modifications.
    
    03. ___________(Integrated development environment/Package manager) is a tool that automates the process of installing, updating, configuring, and removing software packages on a computer system.
    
    04. In addition to software installation and updates, some package managers also provide features for ____________(version/dependency) management, allowing users to rollback to previous versions of installed packages if needed.
    
    05. The purpose of a ____________(IDE/VCS) is to enable collaboration among developers by allowing them to work on the same codebase simultaneously.
    
    Answers
    01. IDE
    02. Git
    03. Package manager
    04. version
    05. VCS

    04. GitHub Projects is a feature used for managing code versioning and repositories.
    
    05. Rebasing is a way to integrate changes from one branch into another by moving the entire commit history of the branch being rebased on top of the target branch.
